    • Information

    The MU puzzle is a puzzle presented by Douglas Hofstadter in his famous book "Gödel, Escher, Bach". The setup is simple:

    • There are three symbols, M, I and U.
    • These symbols can be combined together to make strings of symbols, like MUIUI or MUU.
    • You start with the string MI, and the goal is to turn it into the string MU by applying some sequence of the following rules:
      • Rule 1: Add a U to the end of any string ending in I.
      • Rule 2: Double the string after M.
      • Rule 3: Replace any III with a U.
      • Rule 4: Remove any UU.

    Solution

    It's impossible to solve the puzzle. To do this, you can look at an "invariant", something that doesn't change while applying the rules.

    Consider the total number of Is in a string. Only rule 2 and rule 3 can change this number. Rule 2 will double it while rule three will reduce it by three. An invariant property is that the number of Is is not divisible by 3: at the start, there is one I, which is definitely not divisible by 3. Doubling a number not divisible by 3 does not make it divisible by 3, and subtracting 3 from a number that is not divisible by 3 does not make it divisble by 3 either. But if we were to reach MU, then we would have zero Is, which is divisible by 3, so it's impossible to find a sequence of rules that reach that state.
